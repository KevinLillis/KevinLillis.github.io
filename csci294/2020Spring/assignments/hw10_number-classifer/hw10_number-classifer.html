Homework 10 – Number Classifier

Write a C++ program that does the following:

  1. Prompts the user asking how many number they would like to enter
  2. Prompts the user for each numbr and reads in that number
  3. Displays how many of the numbers were:
    • Odd
    • Even
    • Positive
    • Negative
    • Zero

Required Functions

You program must be designed to use the following functions:

  1. A void function named classifyNumber that has six parameters. The first parameter is of type int and contains the number to be classified. The other five parameters are all int references. One counter for each of the classifications (odd, even, positive, negative, and zero). This function should increment the appropriate counter(s) based on the value of the number.
  2. A void function named printResults that has five int parameters, one counter for each of the classifications (odd, even, positive, negative, and zero). This function should print the classification counts as shown in the sample run below.
  3. The main function should:
    • Initialize the five counters to zero
    • Prompt and read how many numbers will be entered
    • Contain a loop to read each number and call the classifyNumber function to update the counts
    • Call the printResults function

Sample Run

Below is an example of how your program should look when it runs. The user input is shown in green. You program does not need to use any colors. Other than that though, your program should look like the sample runs. 


     How many numbers would you like to enter? 5
     Enter a number: -2
     Enter a number: -1
     Enter a number: 0
     Enter a number: 1
     Enter a number: 2

     Odds      : 2
     Evens     : 3
     Positives : 2
     Negatives : 2
     Zeros     : 1
