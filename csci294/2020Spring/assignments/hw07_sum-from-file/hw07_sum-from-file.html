Homework 07 – Sum From File

Write a C++ program that opens an input text file containing only integers, processes the data, and writes the results to a different text file as well as displays the results on the screen.

Detailed Requirements

  • The program should prompt the user to enter the names of the input and output files.
  • The input and output files are assumed to be in the default location.
  • The input file contains multiple pairs of integers, one pair per line.
  • For each pair of integers x, y, the program is to compute the sum of the numbers from x to y, inclusive.
  • The value of x may be less than, equal to, or greater than the value of y.
  • There should be one line of output for each pair of numbers in the input file.
  • Each line of output is to be displayed on the screen as well as being written to the output file.
  • Each line of output is to indicate the value of the two numbers (smaller first) and the sum. See the sample run below.

Sample Run

Suppose the input file is named input.txt and contains the following:

1 100
100 1
4 4
-8 8
3 -3
-2 -4
-4 -2

Then a sample run of the program would produce the output shown below as well as writing the results to the file named output.txt. The things that the user types in are shown in green. You program does not need to use any colors. Other than that though, your program should look like the sample runs. 


                 Please enter name of input file:  input.txt
                 Please enter name of output file:  output.txt
                 Sum from 1 to 100 is 5050
                 Sum from 1 to 100 is 5050
                 Sum from 4 to 4 is 4
                 Sum from -8 to 8 is 0
                 Sum from -3 to 3 is 0
                 Sum from -4 to -2 is -9
                 Sum from -4 to -2 is -9
