Homework 03 – Reverse Digits

Write a C++ program that prompts the user for an integer and then displays the integer with the digits, and possibly the negative sign, in reversed order.

Sample Runs

Below are three examples of how your program should look when it runs. The things that the user types in are shown in green. You program does not need to use any colors. Other than that though, your program should look like the sample runs. 


                 Enter an integer: 456789
                 987654
         
             


                 Enter an integer: 12300
                 00321
         
             


                 Enter an integer: -45600
                 00654-
