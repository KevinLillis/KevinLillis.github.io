Homework 05 – Recursion

For this assignment you are to write three separate C++ programs as described below.

  1. In class we wrote a recursive function to compute n!. Write an iterative (non-recursive) function to compute n!
  2. Write a recursive program to comute b raised to the e power. For example, 34 = 81
  3. Write an iterative function to compute be

What to Turn In

  1. Name your three C++ source files as follows: hw05-1.cpp, hw05-2.cpp, and hw05-3.cpp.
  2. Create a zip file containing all three of your source files.
  3. Submit the zip file through Blackboard.
