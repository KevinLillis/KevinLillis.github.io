Homework 06 – Guessing Game

In class we wrote a Guessing Game program. For this assignment you are to write an enhanced Guessing Game program.
  1. You program should only accept guesses that are from 1 to 100, inclusive.
  2. At the end of a game the program should show how many tries it took guess the number. Notice that in the example programs provided below the message for 1 guess is slightly different than for more than 1 guess.
  3. At the end of a game the program should rate the player based on the how many tries it took to guess the number. Invalid guesses are not be counted.
    1. 1 - 3 "Lucky"
    2. 5 - 6 "Awesome"
    3. 7 - 9 "Good"
    4. 10 - 12 "Meh"
    5. 13 - 15 "Poor"
    6. More than 15 "Pathetic"
  4. When a game ends, the program should ask if the user wants to play again. They should enter 1 to play again or 2 to quit. The program should only accept entries of 1 or 2.
  5. When the player quits a "Thank you for playing..." is displayed.

Example Programs

Below are two compiled executable programs (one for PC and one for Mac) that demonstrate exactly how your program is to behave.

  • PC Windows: GuessingGame.exe
  • Mac OS X: GuessingGame.o

Required Functions

Your program is to include the following functions:

int getGuess(string prompt)

Displays the given prompt, which should include instructions to enter a number from 1 to 100. Reads a number from the keyboard. Performs input validation to ensure a number from 1 to 100 is entered. Returns the number once it has been accepted.

string getRank(int guessCount)

Given the number of tries it took to guess the secrete number, returns one of the following rankings:

    		    1 -  3 Guesses: "Lucky"
    		    4 -  6 Guesses: "Awesome"
    		    7 -  9 Guesses: "Good"
    		   10 - 12 Guesses: "Meh"
    		   13 - 15 Guesses: "Poor"
    		      > 15 Guesses: "Pathetic"

bool getPlayAgain(string prompt)

Displays the given prompt, which should include instructions to enter to enter 1 to play again or 2 to quit and reads a number from the keyboard. Performs input validation to ensure only 1 or 2 is entered. Returns true or false based on the number entered.

void playOneGame()

Plays one round of the Guessing Game.

int main()

The main function should:

  1. Seed the random number generator
  2. Contain a loop that repeatedly plays the game until the user decides to quit
  3. Print the "Thank you for playing..." message.
