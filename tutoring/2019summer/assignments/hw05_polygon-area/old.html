HW 04 - Area of Regular Polygon

Write a program to find the area of an n-sided polygon in which each side has length a (see picture to the right). Such a polygon is call an n-regular polygon. The area of an n-regular polygon is given by the formula shown below.

Write a program named PolygonArea that prompts the user for the number of sides and the side-length, then prints the area of the polygon. Your program should use the constant math.pi for the value of π and the method math.tan for the tangent function. Save your program in a file named PolygonArea.py.
