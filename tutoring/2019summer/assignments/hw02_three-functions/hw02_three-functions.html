HW 02 — Three Functions

Write the following three functions:

  1. The First Function
    • Name is power
    • Takes two numbers, x and y
    • Returns xy
  2. The Second Function
    • Name is subtract
    • Takes two numbers, x and y
    • Returns x - y
  3. The Third Function
    • Name is smaller
    • Takes two numbers, x and y
    • Returns the smaller of x and y
