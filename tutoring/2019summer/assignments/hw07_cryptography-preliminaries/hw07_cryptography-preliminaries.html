HW 07 — Cryptography Preliminaries

Next week we will start looking at cryptography. Our cryptography program will eventually have two functions, called encrypt and decrypt. To start things off, I'd like you to write a preliminary version of the encrypt function. Here are the details:

  • The function should be called encrypt
  • The function should take the following two parameters:
    1. message, this will be a string, it's what we want to encrypt.
    2. offset, this is just an integer (a whole number) that we'll use to do the encryption.
  • Inside the body of the function, you should create a variable called alphabet that is a string containing the following:
    • The lower case letters a to z
    • The upper case letters A to Z
    • The digits 0 to 9
    • The punctuation symbols . , ? :
