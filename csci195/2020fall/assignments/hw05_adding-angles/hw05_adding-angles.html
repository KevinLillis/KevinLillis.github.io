HW 05 – Adding Angles

Angles are often measured in degrees (°), minutes ('), and seconds ("). There are 360 degrees in a circle, 60 minutes in one degree, and 60 seconds in one minute. Write a program that reads from the keyboard two angular measurements given in degrees, minutes, and seconds and then calculates and prints their sum. You can use the Angle Calculator at https://www.mathopenref.com/dmscalculator.html to verify your program is working correctly.

The output of your program should match the Sample Run shown below. Note, to print the degree symbol in Python you can use the unicode symbol like this: degree_sign = u'\N{DEGREE SIGN}'

Your program should be organized and utilize a main() function like the programs we have written in class.

Sample Run

Note that the user input is shown in green only to make it easier to see how the program behaves. Your program does not need to produce any colors.


    ANGLE 1
        Degrees: 74
        Minutes: 29
        Seconds: 13
    
    ANGLE 2
        Degrees: 105
        Minutes: 8
        Seconds: 16
    
    74° 29' 13"  +  105° 8' 16"  =  179° 37' 29"
