HW 10 – Pi Approximation

Below is yet another formula that we can use to approximate the value of 𝝅. Write a Python function that takes as an argument the number of terms, n, and returns the approximation of 𝝅 given by the first n terms of the series. Your program should also include a main function that prompts the user for the number of terms and then displays the resulting approximation for 𝝅.

Sample Run

Note that the user input is shown in green only to make it easier to see how the program behaves. Your program does not need to produce any colors.


    Enter number of terms: 10000
    pi approximation is 3.1415926535897922

Program Organization

Your program should be organized and utilize a main() function like the programs we have written in class. You should also have a separate function that calculates the approximation of 𝝅.

What to Turn in

Submit your single Python program file ending with a .py file extension.
