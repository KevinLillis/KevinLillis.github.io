HW 01 – Some Good Songs

The traditional first program for anyone learning a new programming language is the Hello World program. This program simply prints the message Hello World to the screen. The purpose of a Hello World program is to make sure you have everything installed correctly to create and run a program. The problem is, the Hello World program is really boring! So we're going to do something different.

Assignment

In November my wife Marie and I will have been married for 35 years and I would like to make her a playlist. Below are a few songs I thought would be good, along with YouTube links. For this assignment I want you to write a Python program that prints the names of a couple of songs you think we would like (not one of the ones listed below). You should also print the name of the artist and a YouTube or Spotify link.

Extra Credit

If your song makes it to our playlist, you'll get extra credit. It might help if you listen to the songs listed below.

Some Songs We Like
