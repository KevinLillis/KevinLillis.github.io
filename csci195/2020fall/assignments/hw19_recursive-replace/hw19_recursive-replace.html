HW 19 – Recursive Replace

For this assignment you will write a single recursive function.

  • The name of the function must be: replace
  • The name of the Python file must be: recursion.py
The recursion.py file should contain no additional functions, including no main() function.

Details about the replace function

The general idea is that the function goes through a list L and replaces all occurrences of the value s with the value t.

The function must be named replace. It takes three parameters:

  1. L: The list
  2. s: The item that will be replaced
  3. t: The new item that will replace s

The function should use recursion to return a new list consisting of all the elements of L, only with every occurrence of s replaced with t.

For example, the function call replace([1, 2, 3, 2, 1], 2, 'dog') would return the list [1, 'dog', 3, 'dog', 1].

What to Turn in

Submit your single Python program file ending with a .py file extension.
