Homework 15 - Quiz Grader

Introduction

For this assignment you will write a program that automatically grades true/false quizzes. Your program will read from an input file with a specific structure (described below) and produces an output file containing the individual quiz scores and a summary for the class. The structure for the output file is also described below.

The Input File

Information about the quiz and the answers given by each student are contained in a text file. The program should prompt the user to enter the name of the text file. The text file will have the following structure:

  • The first line of the file contains a brief description of the quiz. For example something like "Section A - Quiz # 5 - Covers Chapter 2".
  • The second line of the input file contains a single integer m that tells how many questions are on the quiz.
  • The third line contains the correct answer for each question. This will consist of m characters, all separated by a single space. Each character is either T or F.
  • The fourth line contains a single integer n that tells how many students are in the class.
  • This is followed by two lines for each of the n student:
    • The student's ID, which contains no spaces.
    • The student's answers which are all separated by a single space. There are m answers for each student. Each answer is either T, F, or the word blank, spelled out.

Example Input file

Below is an example input file for a quiz consisting of three questions given in a class with five students. You can download this file to test your program: ScoresAssignment.txt.

    Section A - Quiz # 5 - Covers Chapter 2
    3
    T F T 
    5
    GHBT339294
    T F T
    ts3390493
    T blank T 
    Baker123
    blank blank F
    Steiner
    T F T
    2983
    T F T

The Output File

The program produces an output file containing the results of the exam. The name of the output file is based on the name of the input file. The output file has the same name as the input file, only with "results-" added to the front. The structure of your output file must be as follows:

  • The first line of the output file contains the quiz description that was read in from line 1 of the input file.
  • This is followed by one line for each student, consisting of the student's ID, followed by a colon, a single space, and the number of correct answers.
  • At the end of the output file there is one additional line containing the class average. This line contains the words Class Average, followed by a colon, a single space, and the average number of correct answers, which is a floating point number.

Example Output File

Below is the output file produced when the program is executed with the example input file shown above. You can also download this file: results-ScoresAssignment.txt

    Section A - Quiz # 5 - Covers Chapter 2
    GHBT339294: 3
    ts3390493: 2
    Baker123: 0
    Steiner: 3
