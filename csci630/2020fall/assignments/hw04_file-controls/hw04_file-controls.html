HW 04 – Local File Controls

Due Date: Sun 10/18

This assignment explores the access control mechanisms used to protect local files and folders. It consists of two parts: A short paper and a step-by-step activity.

Paper – OS Differences

Prepare a two page summary comparing the file structures and access control mechanisms for two different operating systems. You should compare Windows and one other OS (Mac, Unix, Linux, etc.)

Step-By-Step Activities

Below are just a few brief activities aimed at exploring your own OS. Complete each step and keep detailed notes, including screen-shots as appropriate. Feel free to perform additional activities that illustrate points from your summary above. Submit your notes through Blackboard.

  1. Search the file system on your computer.
    Locate the root folder. From there, locate the following files and provide the full path and file name for each:
    1. The word-processing program you typically use.
    2. A text or word-processing file in your documents directory.
    3. A file containing your preferences for a program like a web browser or word processor.
    4. A file you downloaded from teh internet using your browser.
  2. Can you create files in the root directory of your system? If so, create a file or folder and then delete it. If the attempt fails, describe the error displayed.
  3. Determine whether the system you use implements an isolation policy or a user-sharing policy. Describe the steps you took to verify your conclusion.
  4. Determine whether the system you use will allow a file's owner to block access to an owned file. Create a file and try to remove the owner's Read or Write permission from that file. Did the system remove the right or not? How could you tell?
