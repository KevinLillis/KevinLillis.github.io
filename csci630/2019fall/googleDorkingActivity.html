Google Dorking In-Class Activity

Perform Google searches using various search operators to see if you can find documents that might be sensitive in nature. By selecting different operators and combining them together you can see Google hacking can be effective in exposing sensitive materials. Start by looking at this article: Exploring Google Hacking Techniques Here are some examples from the article: 

  • allintext:username filetype:log myBB sql
     Exposes user names and passwords 
  • inurl:/proc/self/cwd
     Vulnerable server results will appear, along with their exposed directories that can be surfed from your own browser. 
  • intitle:"index of" inurl:ftp
     Lists public FTP servers 
  • filetype:log username putty
     The PUTTY SSH client on windows always logs username. This shows them. 
  • filetype:xls inurl:"email.xls"
     Look for email lists
