Homework 06 — MergeSort

In class we discussed a version of MergeSort that uses Θ(1) extra memory. In other words, this version of MergeSort does not make copies of the array elements. Instead, it includes indexes to specify which part of the array are being looked at. For this assignment you are to write a program that implements this version of MergeSort.

Programming Language

For this assignment you may use any of the following programming languages:

  • Python 3
  • Java
  • C++

What to Turn In

Create a zip file containing the source code file for your program and turn in the zip file through Blackboard.
