Homework 01 — Recursive GCD

In class we discussed Euclid's GCD algorithm and wrote a small program to calculate the GCD of two numbers. For this assignment you are to write a Recursive GCD program. Your program must:

  1. Prompt the user for two numbers
  2. Use a recursive version of Euclid's GCD algorithm to determine the GCD for the two numbers given
  3. Display the GCD
  4. Match the formatting shown in the Sample Run below

Programming Language

For this assignment you may use any of the following programming languages:

  • Python 3
  • Java
  • C++

Sample Run

Below is an example of how your program should look when it runs. The things that the user types in are shown in green. You program does not need to use any colors. Other than that though, your program should look like the sample run (with different numbers, of course). 


             Enter first number: 161622
             Enter second number: 124968
             GCD (161622, 124968) = 246

What to Turn In

Create a zip file containing the source code file for your program and turn in the zip file through Blackboard.
