Homework 09 — Rectangle Class

For this assigment you are to write a class named Rectangle. The details of the class are provided below.

Instance Variables

Your class should have exactly two instance variables, both of which are of type int. The names of the variables should be width and height.

Constructors

Your class is to have two constructors:

public Rectangle()

This is the default constructor. It sets the width and height both to the default value of 1.

public Rectangle(int initialWidth, int initialHeight)

This constructor provides initial values for both of the instance variables. If either value is less than or equal to zero, both the width and the height should be set to the default value of 1.

Methods

Below are the method signatures along with descriptions for each method that you must include in your Rectangle class.

public int getWidth()

Returns the value of the instance variable width.

public int getHeight()

Returns the value of the instance variable height.

public void setWidth(int newWidth)

Sets the instance variable width to the specified value. If the value provided is less than or equal to zero, then the width of the rectangle should not be changed.

public void setHeight(int newHeight)

Sets the instance variable height to the specified value. If the value provided is less than or equal to zero, then the height of the rectangle should not be changed.

public void set(int newWidth, int newHeight)

Sets both instance variables to the specified values. If either value provided is less than or equal to zero, then neigher the height nor the width of the rectangle should be changed.

public int area()

Calculates and returns the area of the rectangle.

public int perimeter()

Calculates and returns the perimeter of the rectangle.

public boolean isSquare()

Returns true if the rectangle is a square and returns false if it is not a square.

public String toString()

Returns a string containing information about the Rectangle object. For example, if the width is set to 2 and height is set to 3, then this method should return the string: Rectangle [width: 2, height: 3]
