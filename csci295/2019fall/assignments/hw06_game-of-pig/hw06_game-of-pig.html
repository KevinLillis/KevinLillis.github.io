Homework 06 — Game of Pig

Pig is a simple one-player dice game were a player tries to accumulate as many points as possible. The game consists of five turns. The final score is the sum of the scores from each turn.

During each turn, the player repeatedly rolls the die until either a 1 is rolled or the player decides not to roll any more for that turn.

  • If a 1 is rolled, the player gets zero points for the turn and the turn immediately ends.
  • It the player rolls any other number, it is added to their total and the player has the option whether or not to roll again for that turn.

At the end of the game, the players score is evaluation as follows:

  • Over 50: Excellent
  • 31 – 50: Average
  • 0 – 30: Poor

Assignment

Write a Java program that allows the user to play a game of Pig. For this assignment you are provided with the skeleton of a Java class that contains sevaral methods. You are required to start with this class and complete the methods as described below. Here is the class to download: Pig.java.

Method Descriptions

public static void main(String[] args)

Your program should include this method exactly as it is. Do not change this method.

public static String evaluateScore(int score)

Once the game game is finisehd, the score is passed to this method. The method evaluates the score and returns one of the following: excellent, average, poor.

public static int playGame(Random gen, Scanner input)

This method contains the main game and includes a loop that executes five times, once for each turn. The body of the loop should include, among other things, a call to the playTurn method. This method should return the final score for the game.

public static int playTurn(Random gen, Scanner input)

This method is responsible for playing a single turn and returning the points earned for that turn.

public static void printWelcome()

This method simply prints the welcome message at the start of the game. See the Sample Run below for what the message should say.

public static boolean rollAgain()

This method is used to determine if the player wants to roll again or end a turn. It returns true to roll again and return false to end the turn.

The syntax for reading a String using a Scanner named in is: in.next().

The syntax to see if a String named value equals "y" is: value.equals("y").

public static int rollDie(Random gen)

This method uses the random number generator to return a value from 1 to 6, inclusive.
