Homework 14 — Array List of Boolean

For this assignent you are provided with an interface called ListOfBoolean. You are to write a class called ArrayListOfBoolean that implements the interface. You should download the interface, save it in your NetBeans project, and then begin working on implementing the interface.

In addition to implementing the ListOfBoolean interface, your class must also include the following:

  • A static int constant that is public. The name of the constant should be DEFAULT_CAPACITY and its value should be 4.
  • A default constructor that uses the default capacity.
  • A constructor that specifies the initial capacity.

Private Instance Variables

Your ArrayListOfBoolean class should include the following private instance variables:
  • An array of type Boolean[]
  • An int variable that holds the size of the list

Implementation Details

Your class is to store the list items in the Boolean array. You should follow the strategy used in the ListOfStrings class we wrote in class.

Create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
