Homework 01 — Some Good Songs

The traditional first program for anyone learning a new programming language is the Hello World program. This program simply prints the message Hello World to the screen. The purpose of a Hello World program is to make sure you have everything installed correctly to create and run a program. The problem is, the Hello World program is really boring! So we're going to do something different.

Assignment

Below is a list of some songs I like along with YouTube links. For this assignment I want you to write a Python program that prints the name of a song you think I will like (not one of the ones listed below), along with the artist and YouTube link.

Extra Credit

If I like your song enough to add it to my Spotify play list, you'll get extra credit. It might help if you listen to some songs from the list below.
