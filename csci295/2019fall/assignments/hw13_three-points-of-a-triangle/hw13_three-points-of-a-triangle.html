Homework 13 — Three Points of a Triangle

In this assignment you will create a class implements an interface. The class holds three points, (x0, y0), (x1, y1), and (x2, y2), that represent the three points of a triangle. The class will provide various methods that have to do with triangles. In addition to creating the class itself, you must also provide a tester program that thoroughly tests all methods and constructors of your class. The class must be called ThreePoints and must implement the interface ThreePointsInterface.java.

Detailed Requirements for the Class

Name of Class and Instance Variables

Create a class called ThreePoints that holds six floating point numbers representing the coordinates of three points. You can think of the three points as (x0, y0), (x1, y1), and (x2, y2). The only instance variables for this class are six floating point numbers, named something like x0, y0, x1, y2, x2, y2.

Constructors

This class should have two constructors. The default (no-argument) constructor should set all instance variables to zero. The second constructor should take six parameters corresponding to the six instance variables.

A Helper Method You Will Need

Because values are calculated using floating point numbers, rounding errors may occur. For example, a right angle may me stored as 89.999999999999. A Boolean expression that tests to see if this angle is equal to 90 would return false. To get around this problem you should use the method approxEqual that we'll discuss in class. This method takes two double values and returns true if the values are approximately equal and returns false otherwise. Here is the method: 


        /**
         * Tests whether two floating-point numbers are approximately equal.
         */
        private boolean approxEqual(double x, double y) {
            return Math.abs(x - y) <= 1E-12;
        }

boolean isEquilateral()

Returns true if the three points form a triangle and that triangle is equilateral, returns false otherwise. A triangle is an equilateral triangle if all three sides are of equal length.

boolean isIsosceles()

Returns true if the three points form a triangle and that triangle is isosceles, returns false otherwise. A triangle is an isosceles triangle if at least two sides are of equal length.

boolean isScalene()

Returns true if the three points form a triangle and that triangle is scalene, returns false otherwise. A triangle is a scalene triangle if all three sides are of unequal length.

boolean isAcute()

Returns true if the three points form a triangle and that triangle is acute, returns false otherwise. A triangle is an acute triangle if all three angles are acute angles (less than 90°).

boolean isObtuse()

Returns true if the three points form a triangle and that triangle is obtuse, returns false otherwise. A triangle is an obtuse triangle if one of the angles is an obtuse angle (greater than 90°).

boolean isRight()

Returns true if the three points form a triangle and that triangle is a right triangle, returns false otherwise. A triangle is a right triangle if one of the angles is a right angle (equal to 90°).

double getPerimeter()

Returns the perimeter of the triangle as a double. If the three points do not form a triangle, this method should return zero. The perimeter of a triangle is equal to the sum of the three sides.
