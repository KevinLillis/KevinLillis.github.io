Homework 10 — Array Methods

For this class you are to write a program that has a main method and the following four additional methods:

  1. getMax – This method returns an int and has one parameter, which is an array of int values. This method should return the largest value in the array.

  2. getMin – This method returns an int and has one parameter, which is an array of int values. This method should return the smallest value in the array.

  3. getMaxIndex – This method returns an int and has one parameter, which is an array of int values. This method should return the index of the largest value in the array.

  4. getMinIndex – This method returns an int and has one parameter, which is an array of int values. This method should return the index of the smallest value in the array.

Create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
