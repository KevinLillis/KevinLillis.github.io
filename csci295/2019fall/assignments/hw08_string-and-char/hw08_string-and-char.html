Homework 08 — String and Char

For this assignment you are to write a program that includes the following methods:

  • static int countVowels(String s)
    This method takes a String parameter and returns the number of vowels in the string.
    For this method, vowels are a, e, i, o, u, A, E, I, O, and U.

  • static void countVowelsTester()
    This method should test the countVowels method by calling it with different strings and printing the results.

  • static int addDigits(String s)
    This method takes a String parameter and returns the sum of any digits that may be in the String.
    So if the string is "A-B-C, 1-2-3" the method should return 6.
    If the string is "St. Ambrose University" the method should return 0.

  • static void addDigitsTester()
    This method should test the addDigits method by calling it with different strings and printing the results.

Your program's main method should simply call the two tester methods.

Create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
