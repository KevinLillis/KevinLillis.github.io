Homework 12 — Part Number Comparator

T he Aircology Inc. company manufactures industrial air quality equipment. The part numbers used by Aircology never contain spaces. But the company uses special symbols as part of their part numbering scheme. The special symbols are #, $, and %. These symbols are used to specify five different Categories of part numbers:

  1. Numbers that contain one or more # symbol, but neither of the other two special symbols.
  2. Numbers that contain one or more $ symbol, but neither of the other two special symbols.
  3. Numbers that contain one or more % symbol, but neither of the other two special symbols.
  4. Numbers that contain more than one of the three special symbols.
  5. Numbers that contain no special symbols.

You have been asked to write a program that reads a list of part numbers from a text file (one part number per line) named partNumbers.txt and print them in sorted order as follows:

  • Part numbers in Category i should come before numbers in Category j whenever i < j.
  • Numbers that are in the same Category should be sorted alphabetically, ignoring all special symbols (#, $, %).

For Example, if the input file contains the following par numbers: 


            bac##def
            ab%cd#ei
            aba$dfe
            a%bcdef
            ab$cd%eg
            abwdef%%%
            bac##def
            a#bcdxf
            $$abcdef$
            ab#cd$ek
            zbcdef
            vbfdec
            ab#cd$ef%%

Your program should generate the following output: 


            a#bcdxf
            bac##def
            bac##def
            aba$dfe
            $$abcdef$
            a%bcdef
            abwdef%%%
            ab#cd$ef%%
            ab$cd%eg
            ab%cd#ei
            ab#cd$ek
            vbfdec
            zbcdef

Hint

All of the part numbers can be read in as Strings and put into a Java ArrayList. You can then sort the ArrayList by providing your own Comparator.

Create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
