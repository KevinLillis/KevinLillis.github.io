Homework 05 — Random Method

Write a Java program that has a methot that:

  • Is named randomNumber
  • Returns an int value
  • Takes two int parameters, called low and high
  • Returns a random integer in the range low ≤ number ≤ high.
    • This is not the same way the nextInt() method from the Random class works.
    • Note that low may be larger than high, or they may have the same value.
    • The random number generator should be seeded with the value 1985.

Your program's main method should:

  • Read two integers from the user
  • Pass the two numbers to the randomNumber method
  • Print the value returned
