Homework 11 — Clock Interface

For this assignment you are to write two separate classes, each of which implements an interface named Clock. Your two classes are to be called Clock_1 and Clock_2. The Clock interface contains the requirements for the public methods. You can also look at the web pages generated by JavaDoc. You should download the Clock interface and copy it into the src directory of your NetBeans project folder.

Instance Variables

The Clock_1 class is to store the hours, minutes, and seconds in three separate instance variables of type int.

The Clock_2 class is to have only one instance variable of type int. This variable holds the number of seconds since midnight.

Constructors

Both clock classes are to include the following constructors. Constructors 1, 2, and 3 must function by calling constructor 4.

  1. A default constructor that sets the time to 0:0:0.
  2. A constructor that takes a single int parameter, hour, and sets the time to hour:0:0.
  3. A constructor that takes two int parameters, hour and minute, and sets the time to hour:minute:0.
  4. A constructor that takes three int parameters, hour, minute, and second, and sets the time to hour:minute:second.

Methods

The Clock interface contains the specifications for the required methods. Make sure both of your clock classes implement the Clock interface.
