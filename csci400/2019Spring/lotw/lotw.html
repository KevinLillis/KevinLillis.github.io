Presentation Guidelines

Your Langgugae of the Week presentation should include the following:

  1. History and Overview
    1. Who developed the language
    2. When was the language developed
    3. What was the motivation for the language
    4. In what domain is the language currently being used
    5. How popular is the language
    6. What is the outlook for the future of the language
  2. Instructions for obtaining and installing the langauge
  3. Overview of the basic syntax for the language
  4. Live coding examples
    1. Write the "Inspirational Quote" program from the Common Assignments
    2. Write the "Pythagorean Triples" program from the Common Assignments
    3. Write other programs to illustrate features of your language (optional)
  5. Describe and explain the programming assignment you developed for the rest of the class.
