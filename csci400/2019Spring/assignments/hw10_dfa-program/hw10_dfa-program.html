Homework 10 — DFA Program

In this assignment you will write a program to implement DFAs. You may write your program using Java or Python.

Detailed Requirements

Your program is to open and read input from a text file named DFA.txt. The text file will contain one or more DFAs. The format of the text file is described below.

  • The first line of the text file contains a single integer. This is the number of DFAs that follow.
  • Each DFA is described as follows:
    • The first line contains a single integer m. This is the number of states in the DFA.
    • The next line contains a string of n characters, containing the alphabet Σ for the DFA. You can assume that each input symbol is a single character.
    • The next line represents the transition function δ. It contains only integers that represent the next states δ(q0,a0), δ(q0,a1), ..., δ(q0,an-1), δ(q1,a0), δ(q1,a1), ..., δ(q1,an-1), ... δ(qm-1,a0), δ(qm-1,a1), δ(qm-1,an-1), for states q0 through qm-1 and input symbols a0 through an-1. The integers on this line are separated by white space.
    • The next line contains a single integer representing the start state.
    • The next line starts with an integer p representing the number of accepting states. This is followed by p integers representing the accepting states. The integers on this line are separated by white space.
    • Next is an arbitrary number of lines containing inputs over Σ. Each input is a string of characters taken from Σ.
    • The lines of input are terminated by a final line containing the string zzz.

For each DFA you are to test all of the inputs and indicate whether each is accepted or rejected. You may not use any regular expression operations built into your programming language.

The output must follow the format shown below.
