In this assignment you will write and submit three separate programs.

Part I — Square Area

Part II — Triangle Area

The area of a triangle in which all three sides have the same length (see picture to the right) is given by the formula

Write a program that prompts the user for the side length and then prints the area of the triangle. Your program should be saved in a file named trianglearea.py.

Part III — Polygon Area

Write a program that prompts the user for the number of sides and the side length, then prints the area of the polygon. Your program should use the constant math.pi for the value of π and the method math.tan for the tangent function. Save your program in a file named polygonarea.py.
