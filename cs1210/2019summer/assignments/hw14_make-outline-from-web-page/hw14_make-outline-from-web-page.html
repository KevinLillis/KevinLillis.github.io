HW 14 — Make Outline From Web Page

For this assignment you will write a function that takes two arguments: a URL for a web page and a file name for the output. The function is to open the URL and construct an outline from the HTML code. The outline is to be saved in a text file with the specified name. The outline should include text between any <h1>, <h2>, <h3>, <h4>, <h5>, or <h6> tags. The text from each heading level should be indented properly so that all <h2> tags are indented under their <h1> tag, etc. Each heading level should also be properly numbered, with heading hx having x levels of numbering.

Assumptions

You can make the following assumptions regarding the HTML code being used to construct the outline:

  1. The <hx> and </hx> tags are always on the same line.
  2. The </hx> end tag always appears with no white space.
  3. The heading levels always increase and decrease by at most one.
