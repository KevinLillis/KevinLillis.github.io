HW 06 — Madhava's Pi Approximation

In the 14th century, Indian mathematician Madhava of Sangamagrama discovered the infinite series for pi shown below. Write a Python 3 function that takes the number of terms n, and returns the approximation of pi given by the first n terms in Madhava's series. Your program should also include a main method that prompts the user for the number of terms and then displays the resulting approximation for pi.
