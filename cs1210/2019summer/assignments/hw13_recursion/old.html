HW 11 - Recursion

For this assignment you will write three separate recursive functions, all saved in the same python source file. The three functions are described below

Remove Vowels

This function should be named removeVowels and should take a single string parameter. The function should return a new string containing all characters from the original string that are not vowels. For the purposes of this assignment, the vowels consist of the following letters a, e, i, o, and u. For example, if the original string is 'University of Iowa', then the function should return 'nvrsty f w'

Add Digits

This function should be called addDigits and should take a single integer parameter, n. The function should return the sum of the individual digits in n. For example, if n = 451, then the function should return 10, since 4 + 5 + 1 = 10. This function may not convert any numbers to strings.

Replace

This function should be called replace and should take three parameters: L, s, and t. The function should return a new list consisting of all the elements of L, only with every occurrence of s replaced with t. For example, the function call replace([1, 2, 3, 2, 1], 2, 'dog') should return the list [1, 'dog', 3, 'dog', 1].
