HW 05 — More Spirals

In class we wrote a program that used the Python turtle to draw a square spiral. For this assignment you are write a program that uses the turtle to draw an n-sided spiral instead of a 4-sided spiral. Your program should use the turtle to read the number of sides from the user (you are not to use the input function). Below are pictures of 3-sided and 5-sided spirals.
