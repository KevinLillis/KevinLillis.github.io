Programming Assignment 6

Adding Angles

Angles are often measured in degrees (°), minutes ('), and seconds (").  There are 360 degrees in a circle, 60 minutes in one degree, and 60 seconds in one minute.  Write a program that reads from the keyboard two angular measurements given in degrees, minutes, and seconds and then calculates and prints their sum.  Use the program to verify the following:

74°29'13"  + 105°8'16"  = 179°37'29"
7°14'55"   + 5°24'55"   = 12°39'50"
20°31'19"  + 0°31'30"   = 21°2'49"
122°17'48" + 237°42'12" = 0°0'0"

You can have your program display the degree symbol with the following constant definition

const unsigned char DEG = 248;

Sample Run



ANGLE 1: Enter degrees, minutes, and seconds separated by spaces: 74 29 13
ANGLE 2: Enter degrees, minutes, and seconds separated by spaces: 105 8 16
