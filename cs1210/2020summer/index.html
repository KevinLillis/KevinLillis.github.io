Notes on Homework

After grading HW 01, I have the following comments regarding your programs:

  1. Your programs should have only one main function. As I mentioned in class, your program should be laid out so that it consists of (i) import statements, (ii) function definitions, and (iii) a single call to the main function.
  2. Be sure to include the statement that calls the main() function. This statement looks like this: main() and should be the last line of your program.
  3. There seems to be some confusion about the docstring at the top of your file. Remember, every homework and lab program must start with a docstring. As an example, if I were a student in this class and I created a file for HW 01 named goodSongs.py then the docstring at the beginning of my file would look like this:
    
        '''
        Kevin Lillis
        goodSongs.py
        Homework assignment HW 01
        6/19/2020
        Print's information about some "good songs"
        '''
