HW 14 – Credit Card Check

Buffon's Needle refers to a simple Monte Carlo method for the estimation of the value of 𝝅. The idea is very simple. Suppose you have a tabletop with a number of parallel lines drawn on it, which are equally spaced at 2 inches apart. Suppose you also have a needle which is one inch long. If you drop the needle on the table, you will find that one of two things happens: (1) the needle hits a line, or (2) the needle does not hit a line. The idea now is to keep dropping this needle over and over on the table, and to record the statistics. Namely, we want to keep track of both the total number of times that the needle is dropped on the table (call this d for drops), and the number of times that it hits a line (call this h for hits). If you keep dropping the needle, eventually you will find that the value d/h approaches the value of 𝝅! Here are a few notes I put together describing how this works.

Assignment

Write a Python program that prompts the user for the number of needles to drop and then uses a Monte Carlo simulation that estimates 𝝅 using Buffon's Needle.

Note that this assignment does not require any animation. You just need to use the logic behind Buffon's Needle to estimate 𝝅.
