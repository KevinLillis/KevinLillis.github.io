HW 10 – Pi Approximation

Below is yet another formula that we can use to approximate the value of 𝝅. Write a Python function that takes as an argument the number of terms, n, and returns the approximation of 𝝅 given by the first n terms of the series. Your program should also include a main function that prompts the user for the number of terms and then displays the resulting approximation for 𝝅.
