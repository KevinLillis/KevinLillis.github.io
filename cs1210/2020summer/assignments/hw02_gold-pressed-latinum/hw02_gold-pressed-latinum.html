HW 02 – Gold Pressed Latinum

One bar of Gold-Pressed Latinum is worth 1,587 U.S. Dollars. Write a Python program that prompts the user for the number of bars of Latinum and displays its value in dollars. Below is a sample of what the output of your program should look like.

Note that the user input is shown in green only to make it easier to see how the program behaves. Your program does not need to produce any colors.


     Enter the number of Gold-Pressed Latinum bars: 3
     The value of your 3.0 bar(s) of Gold-Pressed Latinum is 4761.0 dollars.

Grading

Please review the Grading Rubric for Programming Assignments and the Programming Style Guide. I would especially like to draw your attention to the Docstring section which describes the information that must information that should be included at the beginning of your file.
