HW 22 – Rational Class

In this assignment you will create a class that represents rational numbers (ie. fractions). The name of your class must be rational and it must be saved in a file called rationalFile.py. Details are shown below.

  • The numerator and denominator should be stored using integer instance variables. The default rational number is 0/1. There should be no other instance variables.
  • The sign of the rational number is reflected in the sign of the numerator. That means the denominator will always be a positive integer an the numerator will be either positive for a positive rational number, or negative for a negative rational number.
  • If anything other than two int values are used, a ValueError should be raised.
  • If a zero is used for the denominator, a ZeroDivisionError should be raised.

Required Methods

  • __str__ This method should return a string such as 2/3.
  • __init__ This method should check that the proper types (two int values) and proper values (non-zero denominator) are provided. It should also account for the default values and ensure the sign is represented in the numerator as described above.

    • Your Python File

    You are to turn in a single Python file. Your file:

    1. Must be named rationalFile.py
    2. Must include only the class definition for the rationalclass.
    3. Must not include any other classes or functions, including no main()function
    4. Must contain no statements that are outside of the three functions
