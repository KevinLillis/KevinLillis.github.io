HW 21 – Recursion

For this assignment you are to solve three separate problems using recursion.

Your Python File

You are to turn in a single Python file. Your file:

  1. Must be named recursion.py
  2. Must include exactly three methods, which have the following signatures:
    • def removeSubstrings(message, substring):
    • def closest(L):
    • def strDist(message, mark):
  3. Must not include any other functions, including no main()function
  4. Must contain no statements that are outside of the three functions

Problem 1 – Remove Substrings

For this problem, your function is given a message and a substring, both of which are strings. The removeSubstrings function is to return a new string which is a copy of message only with each occurrence of substring removed. Below are some examples of how this should work.

message substring result
abcdef abc def
abcdef def abc
abcdef bcd aef
abcdef abcdef
aabbaabb aa bbbb
aabbaabb a bbbb
abcdef xyz abcdef

Problem 2 – Closest Numbers in A List

For this problem your function is given a list L of sorted integers. The closest function should return a list containing the two numbers that are the closest. If there are more than one pair of closest numbers, the last pair should be returned. You can assume the given list contains at least two numbers. Below are some examples of how this should work.

List L result
[3, 7, 67, 68, 210, 215] [67, 68]
[3, 7, 67, 168, 210, 215] [3, 7]
[3, 47, 67, 168, 210, 215] [210, 215]
[3, 7] [3, 7]
[3, 3, 3, 3, 3, 3] [3, 3]
[1, 2, 3, 4, 5, 6] [5, 6]
[5, 10, 100, 105, 305, 310] [305, 310]
[5, 10, 15] [10, 15]

Problem 3 – String Distance

For this problem your function is given two strings, called message and mark. The strDist function returns the length of the longest substring of message that begins and ends with the string mark. Below are some examples of how this should work.a

message mark result

cat dog cat dog ca cat 11
at dog cat dog cat cat 11
dog cat dog cat 3
dog dog cat 0

ca cat 0
