HW aa – Baby Names

The United States Social Security Administration maintains yearly data that records the popularity of names given to babies. This data has been downloaded and is available in the file names.zip. You should review the data files and the read-me file to familiarize yourself with how the files are named and the format of the data they contain.

To help you test your program, I have provided six files containing test data for the years 2000 – 2005. These files contain a small amount fictitious data. These files can be downloaded here: testdata.zip.

Assignment

Write a Python program that provides a text-based user menu as follows: 

                    Current range: 1880 - 2018
                    Current output file: output.txt
                       1) Specify date range
                       2) Specify output file name
                       3) Most popular names
                       4) When names first appear
                       5) Quit

                       Make selection:

Below is a description of each menu option:

  1. As you can, the default date range is 1880 – 2018. This option allows you to change the start and end years.
    • If the start year is later than the end year, then the two are swapped.
    • If the start year is outside of default range, then 1880 is used for the start year.
    • If the end year is outside of the default range, then 2018 is used for the end year.
  2. All output for this assignment is written to text files. This command lets you specify the name of the output file. By default, the name of the output file is output.txt.
  3. This determines the most popular male name and the most popular female name for each year in the specified range and writes the results to the current output file. The format of your output must match exactly the sample output file popular.txt.
  4. This determines, for each year in the specified range, which names first appeared in that year. The results are written to the current output file. The format of your output must match exactly the sample output file first.txt.
  5. This quits the program.

File Name & Location

For this assignment you will turn in a single python file named babyNames.py.

The data files should be stored in a separate folder called names.
