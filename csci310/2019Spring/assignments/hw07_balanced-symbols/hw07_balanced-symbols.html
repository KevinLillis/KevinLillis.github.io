HW 05 - Balanced Symbols

For this assignment you are to first write a generic stack that uses a linked structure and then use that stack as an instance variable in a class that checks to see if a given string has balanced symbols.

Linked Stack

Create a generic class called LinkedStack that implements the MyStack interface. It uses a singingly linked structure for the stack, so you can use the same SNode class that we used in the linked list assignment. A pop or top operation on an empty stack causes an EmptyStackException to be thrown.

Balance Checker

Your BalanceChecker class should implement the BalanceCheckerInterface interface. It allows the user to add pairs of matching characters and then check strings to see if they are balanced with respect to the pairs of symbols that have been added.
