Homework 03 — Generic Triple

In class we wrote a Java class called Pair that implemented the generic interface called MyPair. For this assignment you are to write a class called Triple that implements the generic interface MyTriple. While you can pattern part of your Triple class after the Pair class, your Triple class must use an array to store the three elements. The way to make a generic array in Java is like this:

E[] arr = (E[])new Object[3];

It would be a good idea for you to write a tester program to make sure your class is working correctly. However, you do not need to turn in your tester for this assignment.

Create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
