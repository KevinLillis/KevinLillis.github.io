HW 09 - Words Part 1

In homework 8 you got the StringGraph class up and running. In this assignment you will write a program to begin using that class. Your program is to construct a graph in which each vertex is a 5-letter word and there is an edge between two vertices if they start with the same letter. Here is a file called wordsABCDEF.txt that contains 5-letter words starting with the letters a-f. Your program is to randomly select eight of these words as the vertices for the graph.

Your program should print the graph using the appropriate toString method in the StringGraph class. A sample output is provided below (you will likely have different words when your program runs).

Your program must include the following methods:

String[] getVertices()
This method takes no parameters and returns a String array containing eight words selected randomly from those provided. This method is responsible for interacting with the input file to select the words.

boolean areNeighbors(String v1, String v2)
This method takes two strings and returns true only if the two strings begin with the same letter.

What To Turn In

Create a single zip file that contains the entire project folder created by NetBeans.
