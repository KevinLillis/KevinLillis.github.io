HW 08 - First String Graph

Part I

Copy this Graph interface into your NetBeans project and implement the interface using the StringGraph class we wrote in class.

Part II

Write a "tester" program that creates a default StringGraph object and then performs the operations listed below. After each operation, your program should display information about the graph. Your output must exactly match the Sample Output shown below. Most of the output can be achieved using the appropriate toStirng method from the StringGraph class. Your program should:

  1. Individually add three vertices labeled "A", "B", and "C" by calling the addVertex method three times.
  2. Add vertices "D", "E", and "F" all at once using the addVertices method.
  3. Remove vertex "B".
  4. Individually add the three edges {"A", "C"}, {"A", "D"}, and {"C", "E"} by calling the addEdge method three times.
  5. Add the edges {"F", "A"}, {"F", "D"}, and {"F", "E"} all at once using the addEdges method.
  6. Remove vertex "A".
  7. Remove edge {"E", "C"}.

Part III

For each operation listed in Part II, hand-draw a picture of the graph.

What To Turn In

For Part I and Part II, create a single zip file that contains the entire project folder created by NetBeans.

For Part III, scan a copy of your graph drawings.

The zip and scanned files should both be turned in through Blackboard.
