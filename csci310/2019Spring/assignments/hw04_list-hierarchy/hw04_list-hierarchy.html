Homework 04 — List Hierarchy

For this assignment you will create a class hierarchy that has the MyList.java interface as its base. A simplified class hierarchy diagram is shown below. A brief description of the interface and classes is provided below. Here is a list of the methods that are required to be included for the interface and each class. You should read the JavaDoc pages for a complete description of the interface, the classes, and the methods.

The class hierarchy will be completed in three parts, each of which will be submitted and graded separately. The three parts are cumulative. In other words, Part II should include everything from Part I, and Part III should include everything from Part I and Part II.

Tester

In order to assist you in determining whether or not your class hierarchy is working properly, I am providing you with a Tester program. You should download this program and copy it into your NetBeans project. The tester thoroughly tests the entire class hierarchy. When working on Part I of this assignment you will need to comment out the parts of the tester for singly and doubly linked lists. When working on Part II you can uncomment the singly linked list tests. When working on Part III you can uncomment the tests for the doubly linked lists.

Grading

When grading your homework I plan to use the same Tester program I provided to you.

Part I - Abstract Class and Array-Based Lists

MyList Interface

This is the interface that you are to download and include in your project. The JavaDoc for MyList provides descriptions of all the methods.

MyAbstractList

This class implements the MyList interface. It includes methods that can be used with all three implementations of MyList. The methods that must be included are described in the JavaDoc for MyAbstractList.

MyArrayList

This class extends MyAbstractList. It provides an array-base list implementation that includes only those methods not implemented in MyAbstractList. The methods that must be included are described in the JavaDoc for MyArrayList.

MyArrayListOp

This class extends MyArrayList and overrides certain methods that are implemented in MyAbstractList for optimization. The specific methods that must be overridden are shown in the JavaDoc for MyArrayListOp.

Part II - Singly Linked Lists

MyLinkedList

This class extends MyAbstractList. It provides a list implementation based on a singly-linked list. It includes only those methods not implemented in MyAbstractList. The methods that must be included are described in the JavaDoc for MyLinkedList.

MyLinkedListOp

This class extends MyLinkedList and overrides certain methods that are implemented in MyAbstractList for improved efficiency. The specific methods that must be overridden are shown in the JavaDoc for MyLinkedListOp.

Part III - Double Linked Lists

MyDLinkedList

This class extends MyAbstractList. It provides a list implementation based on a double-linked list. It includes only those methods not implemented in MyAbstractList. The methods that must be included are described in the JavaDoc for MyDLinkedList.

MyDLinkedListOp

This class extends MyDLinkedList and overrides certain methods that are implemented in MyAbstractList for improved efficiency. The specific methods that must be overridden are shown in the JavaDoc for MyDLinkedListOp.

For each part, create a single zip file that contains the entire project folder created by NetBeans and then submit your zip file through Blackboard.
