Homework 06 — Car Wash Simulation

As we discussed in class, this assignment involves creating a car wash simulation. The simulation should be set up as follows:

  • The car wash takes s minutes to wash a car.
  • The probability that a car arrives during any particular minute is given by p, where 0 ≤ p ≤ 1.
  • The simulation runs for a minimum of d simulated minutes, during which time a car arrives each minute with probability p. After the initial d minutes have elapsed, the simulation continues to run until all cars have been washed.
  • At the end of the simulation a log is printed to the standard output device.

Additional Details

JavaDoc

Here are JavaDoc pages I've generated for this assignment.

Generic Queue

For this part of the assignment you are to write generic queue class as follows:

  • The name of your class must be LinkedQueue.
  • Your class must implement this QueueInterface.java interface.
  • Your class must use a linked structure to store the items in the queue. You can use the SNode class from Homework 04.
  • The instance variables are two SNode references named front and rear and an int named size.
  • The only constructor is the default constructor which sets front and rear to null and sets size to zero.
  • The queue interface requires an EmptyQueueException to be thrown. This is a custom exception that you'll have to write

Car Class

Your simulation will need to have a class to represent each car as an object.

  • The name of this class should be Car.
  • The Car class must implement the CarInterface.java interface.
  • The Car class has three int instance variables: arrivalTime, startWashTime, and id.
  • There is just one constructor for the Car class. It takes initial values for the arrival time and the ID.

public Car(int arrivalTime, int id){ ... }

CarWashSimulator Class

This is the class that actually performs the simulation. It has only one constructor that takes parameters for d, s, and p (see discussion above).

public CarWashSimulator(int duration, int washTime, double arrivalProb){ ... }

This class must implement the SimulatorInterface.java interface.

Note that the CarWashSimulator does not actually print anything while the simulation is running. Instead, it saves the log in an instance variable that can be retrieved using the getLog method once the simulation is complete.

Main Program

Your main program should be in a class called SimRunner. The main method for this program will be very small. It should:

  1. Create a CarWashSimulator.
  2. Call the simulator's runSimulation method.
  3. Print the log returned from the simulator's getLog method.

Required Output and Sample Run
