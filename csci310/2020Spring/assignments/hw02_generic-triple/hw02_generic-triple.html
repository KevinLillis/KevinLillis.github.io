Homework 02 — Generic Triple

This assignment is intended to help you become more familiar with Java generics. You are to write a generic class called Triple that implements the interface MyTriple.

A "Triple" is a set of three Java objects, not necessarily of the same type. For example, a Triple may consist of three Strings, or maybe two Strings and an Integer, or an Integer and two Strings, and so on. The only restriction is that each of the three typs in the triple must implement the interface Comparable.

In addition, notice that the Triple class itself must also implement Comparable. That means your Triple class will need to have a compareTo method. The compareTo method should compare two triples based on their first elements. If the first elements are the same, then it should go on to compare their second elements. If both the first and second elements are the same, then it should look at their third elements.

The basic outline of the compareTo method was covered in CSCI 295, Programming II. You can see an example of a compareTo method in the Coin.java file mentioned below.

Sample Program

Here is a sample program that uses the Triple class. This program creates an ArrayList of Triples. Each Triple consists (i) a String, (ii) an Integer, and (iii) a Coin. The ArrayList is printed, sorted, and printed again.

To run this program you need:

  • TripleDemo.java – The sample program.
  • MyTriple.java – The interface.
  • Coin.java – A coin class that is added to the triples
  • Triple.java – This is the class you need to write. It should not contain a main method.
