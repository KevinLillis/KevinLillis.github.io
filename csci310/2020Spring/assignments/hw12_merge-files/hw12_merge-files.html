Homework 12 — Merge Files

In class we looked talked about how to merge two sorted arrays. This assignment also deals with merging, but instead of merging two sorted arrays, you will be merging an arbitrary number of text files, each containing an arbitrary number of integers, saved one per line in sorted order. The merged results are stored in a text file.

File Names

The first file your program must read is named fileNames.txt. The first line of this file contains the name of the output file where the merged results are to be stored. After that are the names of files containing the integers. There is guarenteed to be at least one file containing data and at least one integer in each data file. Assume all files are in the default location.

Output

Your program is required to produce output to the console and to the output file. On the console, your program should print the message "Merging the following files:", and then list the names of the data files, one per line, followed by the message "Results stored in file_name". See below for an example.

The output file is to contain the merged integers as described above.

Restrictions

  • Your program is not allowed to read all of the numbers ahead of time and save them into arrays or lists.
  • You program is not allowed to merge the numbers into an array or a list. Instead, every time the next number is selected, it must immediately be written to the output file.
  • Your program is not allowed to do any sorting.

Example

Suppose we have the following files: 
     fileNames.txt       file 1.txt        file2.txt        scores.txt
     -------------       ----------        ---------        ----------
     merged.txt          2                 1                4
     file 1.txt          3                 5                6
     file2.txt           5                 7                8
     scores.txt          10                9

The the output to the console would be: 

     Merging the following files:
     file 1.txt
     file2.txt
     scores.txt
     
     Results stored in merged.txt

The file merged.txt file would contain 

    1
    2
    3
    4
    5
    5
    6
    7
    8
    9
    10
