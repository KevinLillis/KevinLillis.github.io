Homework 03 — Generic Array List

For this assignment you are to write the classes in the hierarchy shown below. This hierarchy has as its base the interface MyList.java, which is being provided to you. Here is a list of the methods that are required to be included for the interface and each class. You should read the JavaDoc pages for a complete description of the interface, the classes, and the methods.

Abstract Class and Array-Based List

MyList Interface

This is the interface that you are to download and include in your project. The JavaDoc for MyList provides descriptions of all the methods.

MyAbstractList

This class implements the MyList interface. The methods in this class may not access the array. They can only call methods given in the interface. The methods that must be included are described in the JavaDoc for MyAbstractList.

MyArrayList

This class extends MyAbstractList. It provides an array-base list implementation that includes only those methods not implemented in MyAbstractList. The methods that must be included are described in the JavaDoc for MyArrayList.

MyArrayListOp

This class extends MyArrayList and overrides certain methods that are implemented in MyAbstractList for optimization. The specific methods that must be overridden are shown in the JavaDoc for MyArrayListOp.
