Homework 10 — String Graph, Part 1

In class we discussed the Graph.java interface. For this assignment you will begin to implement this interface.

Instance Variables

Your class should contain the following instance variables:

  • protected String[] labels; // 1D array to store the vertices
  • protected boolean[][] edgeMatrix; // 2D array to store adjacencies between vertices
  • protected int numVertices;
  • protected int numEdges;
  • protected int capacity;

Methods

At this point, we haven't discussed how to implement all of the methods required by the Graph interface. So for this assignment you are only required to implement the methods indicated below. For now, the bodies of the remaining methods can throw the following exception: throw new UnsupportedOperationException("Not supported yet.");. The required methods are:

  • int numberOfVertices()
  • int numberOfEdges()
  • String[] getVertexLabels()
  • void addVertex(string newVertex)
  • boolean vertexExists(String vertex)
  • boolean edgeExists(String vertex1, String vertex2)
  • void deleteVertex(String vertex)
  • void addEdge(String vertex1, String vertex2)
  • void deleteEdge(String vertex1, String vertex2)
  • void addVertices(String[] newVertices)
  • void addEdges(String[][] newEdges)
  • void resize(int newCapacity)

Constructors

Your implementation is required to include the following constructors:

  • public StringGraph()
  • public StringGraph(int initCapacity)
  • public StringGraph(String[] initVertices)
  • public StringGraph(String[] initVertices, String[][] initEdges)

toString() Methods

Your class should contain several "toString" methods, named toString1(), toString2(), etc. There should also an actual toString() methods that simply calls one of the other toString methods. The different required toString methods should:

  1. Show the order, size, maximum degree, minimum degree, and average degree of the graph
  2. Show the order and size of the graph, then list each vertex along with its adjacent vertices
  3. Show the order and size of the graph, then list each vertex in sorted order, along with each vertex's neighbors, in sorted order
  4. Show the adjacency matrix (see sample runs below for details)l

Tester Program

Write a tester program that creates a default StringGraph object and then performs the operations listed below. After each operation, your program should display information about the graph. Your output must exactly match the Sample Output shown below. Most of the output can be achieved using the appropriate toStirng method. Your program should:

  1. Individually add three vertices labeled "A", "B", and "C" by calling the addVertex method three times.
  2. Add vertices "D", "E", and "F" all at once using the addVertices method.
  3. Remove vertex "B".
  4. Individually add the three edges {"A", "C"}, {"A", "D"}, and {"C", "E"} by calling the addEdge method three times.
  5. Add the edges {"F", "A"}, {"F", "D"}, and {"F", "E"} all at once using the addEdges method.
  6. Remove vertex "A".
  7. Remove edge {"E", "C"}.

Hand-Drawn Graphs

For each operation listed under the tester, hand-draw a picture of the graph.
